## 🔧 技术栈规划

### 当前
- **后端**: FastAPI + SQLite + APScheduler
- **前端**: Vue3 + Vite + TailwindCSS + Pinia
- **数据源**: AkShare + Sina + Eastmoney

### 未来扩展
- **缓存**: Redis（可选，提升性能）
- **搜索**: Elasticsearch（全文检索）
- **时序数据库**: InfluxDB（高频行情）
- **图数据库**: Neo4j（企业图谱）
- **消息队列**: RabbitMQ（异步任务）
