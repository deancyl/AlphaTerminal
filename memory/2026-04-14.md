### 重要教训
- Vue 3 模板中访问普通对象包裹的 ref 时**不会**自动解包（只有组件 reactive state 才会）
  - `v-for="acc in store.portfolios"` → `store.portfolios` 是 ref 对象本身，迭代的是空
  - 修复：`v-for="acc in store.portfolios.value"`
- JS 模块模板字符串 URL 不能用反斜杠转义 `\``，要直接用单引号包裹
  - 错误：`fetch(\`/api/.../${id}\`)`
  - 正确：`fetch('/api/...' + id)`
