## 三、前后端代码协同问题

### 3.1 API 路径不一致
| 前端调用 | 后端路由 | 状态 |
|----------|----------|------|
| `/api/v1/sentiment/gauge` | `/api/v1/market/sentiment` | ❌ 路径错误 |
| `/api/v1/portfolio/holdings` | `/api/v1/portfolio/?user=xxx` | ❌ 需传参 |
| `/api/v1/futures/dashboard` | `/api/v1/futures/` | ❌ 路径错误 |

### 3.2 返回格式不统一
```javascript
// market API 返回
{ code: 0, data: {...}, timestamp: xxx }

// portfolio API 返回
{ portfolios: [...] }  // 缺少标准外层

// stocks API 返回
{ data: [...] }  // 无 code 字段
```
