## 一、项目概述

### 1.1 产品定位

本产品是面向机构投顾、专业投资者、金融研究人员的**Web端纯投研型金融终端**，依托全市场、全资产、全周期的金融数据能力，覆盖宏观、行业、公司、基金、债券、商品、外汇等全品类投研场景，**彻底剥离交易属性**，打造合规的投研与投顾服务专属载体。

**核心愿景**：将传统桌面级重度金融数据终端的核心功能迁移至浏览器环境，打破物理设备与操作系统的限制，建立覆盖全球金融市场多资产类别的数据与信息枢纽。

### 1.3 核心约束

1. **永久去除所有交易功能** - 无任何下单、转账、持仓相关入口
2. **合规优先** - 符合《证券法》《证券投资顾问业务暂行规定》
3. **数据源限制** - 使用AkShare（免费开源），无付费数据源
4. **本地化部署** - 完全本地运行，数据不出本地
5. **Web端优先** - 核心功能基于浏览器环境，打破设备限制
