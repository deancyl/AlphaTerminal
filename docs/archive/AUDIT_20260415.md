## API Key 配置说明

export ADMIN_API_KEY="your_secure_key"

# 使用时
curl -X POST http://localhost:8002/api/v1/admin/cache/invalidate \
  -H "Content-Type: application/json" \
  -H "X-API-Key: your_secure_key" \
  -d '{"cache_type": "memory"}'
